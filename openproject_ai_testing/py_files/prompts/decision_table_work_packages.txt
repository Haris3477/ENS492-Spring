=== WORK PACKAGE FIELDS ===

When creating or updating a work package, the request body is JSON with these fields:

- "subject": string (required, must not be empty, maximum 255 characters)
- "description": object with "raw" string field (optional)
- "startDate": string in "YYYY-MM-DD" format (optional)
- "dueDate": string in "YYYY-MM-DD" format (optional, must not be before startDate)
- "estimatedTime": string in ISO 8601 duration format e.g. "PT2H" for 2 hours, "PT30M" for 30 minutes (optional)
- "percentageDone": integer between 0 and 100 inclusive (optional)
- "_links": object containing:
    - "type": object with "href" pointing to a valid type (optional, defaults to Task)
    - "status": object with "href" pointing to a valid status (optional, defaults to New)
    - "priority": object with "href" pointing to a valid priority (optional, defaults to Normal)
    - "assignee": object with "href" pointing to a valid user e.g. "/api/v3/users/1" (optional)
